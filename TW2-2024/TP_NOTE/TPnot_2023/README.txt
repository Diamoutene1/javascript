Le sujet comporte 2 exercices :
- exo 1 noté sur 10 points
- exo 2 noté sur 10 points

ATTENTION : il vaut mieux traiter moins de questions avec un code qui fonctionne plutôt que d'écrire plein
de code qui ne fonctionne pas.

Les questions sont dans les scripts exo1.js et exo2.js, qui contiennent les fonctions que vous aurez à compléter, par exemple :
function toto(/* paramètre(s) à compléter si besoin */) {

}

Vous pouvez travailler en local sans mettre le TP sur votre serveur web, cela est même déconseillé.
Vous devez déposer sur ecampus une archive zip du dossier L1-TW2-2023-tpnote/ avec les 2 exercices
(même si rien n'est fait dans un exo).
