TP NOTÉ TW2 - 10 avril 2024 - 10h-12h

- Vous disposez de 2h pour faire le TP
- Documents autorisés : notes de cours et TP manuscrites, code des TP de TW2 du semestre
- Applications autorisées : site du cours TW2, sites de documentation
- SONT INTERDITS :
   * toute application permettant de communiquer : email, chat, etc
   * sites et applications d'intelligence artificielle ou de génération de code
   * toute application permettant d'obtenir une aide externe (humaine ou non) pour la réalisation du TP

INSTRUCTIONS

- compléter les fichiers .js avec votre code

- créer une archive du dossier complet tw2-tpnote/ et la déposer sur ecampus
  ATTENTION bien mettre tout le dossier même si vous n'avez pas fait certains exercices

- les questions sont dans les fichiers exo1.js, exo2.js et exo3.js, qui contiennent les fonctions
  que vous aurez à compléter, par exemple :
  function toto(/* paramètre(s) à compléter si besoin */) {

  }

- ATTENTION : il vaut mieux traiter moins de questions avec un code qui fonctionne plutôt que d'écrire plein
de code qui ne fonctionne pas.
- travailler en local sans mettre le TP sur votre serveur web, cela est même déconseillé vu que le serveur risquera d'être saturé
